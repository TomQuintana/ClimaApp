## Getting Started
### Installation

1. Generate ApiKey in [Api Weather](https://openweathermap.org/weather-dashboard) and put into APPID_WEATHER inside of .env file

2. Clone the repo
   ```sh
   git clone https://github.com/TomQuintana/ClimaApp.git
   ```

3. Install NPM packages
   ```sh
   npm install

4. Run project 
```sh
    npm run dev
```
